# Sales Trainer

Next.js + Tailwind starter for the Outbound Sales Trainer project. Bootstrapped like the CV Privacy Optimizer stack so we can share patterns (App Router, TypeScript, Tailwind v4).

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` – start dev server
- `npm run build` – production build
- `npm run start` – run the built app
- `npm run lint` – lint the codebase

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
- ESLint (Next config)
